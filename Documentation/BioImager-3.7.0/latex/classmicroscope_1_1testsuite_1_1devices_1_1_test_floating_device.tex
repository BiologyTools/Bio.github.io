\doxysubsection{Detailed Description}
\begin{DoxyVerb}Simple device with a UID after having been initialized.

Floating devices are devices where we can't specify which one to
get, we can only construct it and after initialisation check its
UID.  In this class for test units we can check which UID to get.\end{DoxyVerb}
