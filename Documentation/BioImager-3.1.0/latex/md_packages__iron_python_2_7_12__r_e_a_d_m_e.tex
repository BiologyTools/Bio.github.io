Iron\+Python is an open-\/source implementation of the Python programming language which is tightly integrated with the .NET Framework. Iron\+Python can use the .NET Framework and Python libraries, and other .NET languages can use Python code just as easily.

Iron\+Python can be obtained at \href{http://ironpython.net/}{\texttt{ http\+://ironpython.\+net/}}.
