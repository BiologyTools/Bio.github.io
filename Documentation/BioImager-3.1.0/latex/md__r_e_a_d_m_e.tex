A .NET microscopy imaging application based on \mbox{\hyperlink{namespace_bio}{Bio}}. Supports various microscopes by using imported libraries \& GUI automation. Supported libraries include Prior® \& Zeiss® \& all devices supported by \href{https://github.com/python-microscope}{\texttt{ python-\/microscope.}} If your microscope is not supported check your manufacturer\textquotesingle{}s SDK to implement \char`\"{}\+Microscope.\+cs\char`\"{} or use GUI Automation functions. Works with or without hardware through microscope simulation. Allows for tiled \& depth stack imaging \& supports XInput game controllers to move stage, take images, run ImageJ macros on images or \mbox{\hyperlink{namespace_bio}{Bio}} C\# scripts. Also check out the wiki for \href{https://github.com/BiologyTools/Bio/wiki/Library-Usage}{\texttt{ library usage.}} or check out the \href{https://biologytools.github.io/Documentation/BioImager-2.1.0/index.html}{\texttt{ documentation.}}

\hypertarget{md__r_e_a_d_m_e_autotoc_md966}{}\doxysection{Features}\label{md__r_e_a_d_m_e_autotoc_md966}

\begin{DoxyItemize}
\item C\# scripting with sample tool-\/script and other sample scripts in \char`\"{}/\+Scripts/\char`\"{} folder. \href{https://github.com/BioMicroscopy/BioImage-Scripts}{\texttt{ See samples}}
\item Supports running ImageJ macro commands on images open in \mbox{\hyperlink{namespace_bio}{Bio}}. New Console to run ImageJ macro commands and \mbox{\hyperlink{namespace_bio}{Bio}} C\# functions.
\item Multiple view modes like Emission, and Filtered. ROI\textquotesingle{}s shown for each channel can be configured in ROI Manager.
\item Supports drawing shapes \& colors onto 16 bit \& 48 bit images, unlike System.\+Drawing.\+Graphics.
\item Convenient viewing of image stacks with scroll wheel moving Z-\/plane and mouse side buttons scrolling C-\/planes.
\item Editing \& saving ROI\textquotesingle{}s in images to OME format image stacks.
\item Copy \& Paste to quickly annotate images and name them easily by right click.
\item Select multiple points by holding down control key, for delete \& move tools.
\item Exporting ROI\textquotesingle{}s from each OME image in a folder of images to CSV.
\item Easy freeform annotation with magic select tool which selects based on blob detection.
\item Use AForge filters by opening filters tool window and right click to apply. Currently supports only some AForge filters as many of them do not support 16bit and 48bit images. Convert to 8bit image to make use of more filters. Applyed filters can be easily recorded and used in scripts. \mbox{\hyperlink{namespace_bio}{Bio}} impliments some filters like crop for 16 \& 48 bit images.
