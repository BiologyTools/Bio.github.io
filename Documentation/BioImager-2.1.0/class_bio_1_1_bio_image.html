The function starts by checking if the image is a single channel image. If it is, then the function creates an array of BufferInfo objects, one for each channel. The function then calls the GetEmissionBitmap function of the BufferInfo class to combine the emission images from each channel into a single image. The function then returns the combined image.

◆ MergeZ()

static BioImage Bio.BioImage.MergeZ ( BioImage  b)
static

It takes a 3D image and merges the Z-stack into a single 2D image.

The function takes a BioImage object as an argument. It then creates a new BioImage object, which is a copy of the original BioImage object, but with the Z-dimension set to 1.

The function then loops through the C and T dimensions of the original BioImage object, and for each C and T, it creates a new Merge object, which is a class that is used to merge two images. The first image is the first image in the Z-stack, and the second image is a blank image. The function then loops through the remaining images in the Z-stack, and merges each image with the previous image.

The function then creates a new BufferInfo object, which is a class that contains information about an image. The function then adds the BufferInfo object to the Buffers list of the new BioImage

Parameters
BioImageThe image to be merged
Returns
A new BioImage object.
