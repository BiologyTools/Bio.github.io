Quickly determine whether a whole slide image is recognized.  

If Open\+Slide recognizes the file referenced by {\itshape filename} , return a string identifying the slide format vendor.\+This is equivalent to the value of the Native\+Methods.\+VENDOR property. Calling Open(string) on this file will return a valid Open\+Slide object or an Open\+Slide object in error state.

Otherwise, return {\ttfamily null}.Calling  \+Open(string) on this file will also return {\ttfamily null}.
