\doxysubsection{Detailed Description}
\begin{DoxyVerb}Implements FilterServer wheel interface for Thorlabs FW102C.

Note that the FW102C also has manual controls on the device, so clients
should periodically query the current wheel position.\end{DoxyVerb}
