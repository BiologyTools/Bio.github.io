{\footnotesize\ttfamily void Bio.\+Statistics.\+Mean\+Histogram (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

This function takes the sum of all the values in the stack and divides it by the number of values in the stack.

The stack\+Mean is the average of all the values in the stack.

The stack\+Median is the highest value in the stack.

The stack\+Values is the array of values in the stack.

The count is the number of values in the stack.

The meansum is the sum of all the values in the stack.
