string SciChain.Block.CalculateHash ( )

The CalculateHash function generates a SHA256 hash based on a combination of timestamp, previous hash, and serialized transactions.

Returns
The method CalculateHash is returning a lowercase hexadecimal string representation of the SHA-256 hash value calculated from the concatenation of the TimeStamp, PreviousHash (if not null), and the serialized Transactions in JSON format.
